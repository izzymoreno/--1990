Не забудьте разместить графические файлы в соответсвующих каталогах:
1) Имя диска:\Igor Talkoff Кумир-1990[]\Graphics\SkyJpeg
2) Имя диска:\Igor Talkoff Кумир-1990[]\Graphics\Talkoff
и т.д.
